# Route Finder

The aim of this project is to calculate the shortest possible route across a field with sinkholes.

## Description
The program finds the shortest path from one end of the field to the other while avoiding obstacles such as sinkholes.

## Features
- Shortest path calculation
- Obstacle avoidance
- Grid-based navigation

## Topics Covered
- Graph algorithms
- Pathfinding algorithms
- Optimization problems
- Problem solving
