# Dead Hand Control Logic Simulation

This project models the control logic of the **Dead Hand (Perimeter) system** using Verilog HDL and simulates its behavior.

## Description

The goal of this project is to implement and simulate the decision logic of the Dead Hand protocol to analyze whether escalation can be stopped before reaching the **Point-of-No-Return**.

The system is modeled using digital control logic and simulated to observe its behavior under different conditions.

## Concepts Used

* Control logic modeling
* Digital system simulation
* State-based decision logic
* Verilog HDL
