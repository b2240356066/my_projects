# 5-bit Adder and Multiplier Design

This project implements fundamental arithmetic components using **Verilog HDL** and integrates them into a larger arithmetic unit.

## Description

Individual digital modules were designed and implemented, including multiplexers, multipliers, and 5-bit full adders. These modules were then combined to create a **5-bit Adder/Multiplier unit**.

The project demonstrates modular hardware design and the implementation of arithmetic logic using Verilog.

## Components

* Multiplexer
* Multiplier
* 5-bit Full Adder

## Concepts Used

* Combinational circuit design
* Arithmetic logic implementation
* Modular hardware design
* Verilog HDL
