## Verilog Projects

This folder contains digital design projects implemented using **Verilog HDL**, focusing on hardware description, simulation, and fundamental digital system design concepts.

### Topics Covered
Digital Logic Design  
Combinational Circuits  
Sequential Circuits  
Flip-Flops (D, JK)  
Multiplexers and Arithmetic Units  
Testbench Development and Simulation  
Hardware Modeling using Verilog HDL
