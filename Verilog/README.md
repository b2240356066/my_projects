# Verilog Projects

This folder contains digital design and hardware simulation projects implemented using **Verilog HDL**.

## Projects

### 1. 5-bit ALU and Multiplier Design
Implementation of fundamental digital arithmetic components including:
- Multiplexer
- Multiplier
- 5-bit Full Adder

These components are individually implemented and then integrated to build a **5-bit Adder/Multiplier unit**.

Concepts used:
- Combinational circuit design
- Arithmetic logic implementation
- Modular hardware design in Verilog

---

### 2. Flip-Flop Implementations and Testbenches
Design and simulation of fundamental sequential circuit elements:

- **D Flip-Flop**
- **JK Flip-Flop**

Includes Verilog **testbenches** to verify behavior through simulation.

Concepts used:
- Sequential logic
- Clocked circuits
- Hardware verification using testbenches

---

### 3. Dead-Hand Control Logic Simulation
Modeling and simulation of the **Dead Hand (Perimeter) nuclear control logic** in Verilog.

The project focuses on implementing the system's decision logic and simulating its behavior to analyze whether escalation can be stopped before reaching the **Point-of-No-Return**.

Concepts used:
- Control logic modeling
- System-level digital simulation
- State-based decision logic
