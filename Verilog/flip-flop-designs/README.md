# Flip-Flop Implementations

This project focuses on the implementation and simulation of fundamental sequential circuit elements using **Verilog HDL**.

## Description

Two essential flip-flop circuits were implemented and verified through simulation using testbenches.

## Implemented Circuits

* D Flip-Flop
* JK Flip-Flop

Each flip-flop includes a corresponding **testbench** to validate the circuit behavior under different clock and input conditions.

## Concepts Used

* Sequential circuit design
* Clocked logic
* Hardware verification using testbenches
* Verilog HDL simulation
