# University Library Management System

A Java-based library management system designed using object-oriented programming principles.

## Description
This system simulates a university library environment by managing different user roles such as students, academic members, and guests.
Each user type follows distinct borrowing rules, ensuring a realistic and well-structured system.

## Features
- User role management (students, academic members, guests)
- Borrowing and returning operations
- File-based input processing
- Penalty handling

## OOP Concepts Used
- Inheritance
- Encapsulation
- Polymorphism
