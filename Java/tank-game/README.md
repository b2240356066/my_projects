# Tank 2025

A 2D tank game developed using Java and JavaFX.

## Description
This project implements a GUI-based game using the JavaFX framework.
The game allows user interaction via mouse and keyboard and includes core gameplay mechanics.

## Features
- JavaFX-based graphical user interface
- Player and enemy tanks
- Shooting and collision mechanics
- Score, lives, pause, and game-over screens

## Technologies
- Java
- JavaFX

## Notes
No frameworks other than JavaFX were used
