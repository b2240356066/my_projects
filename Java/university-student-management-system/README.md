# University Student Management System

A student management system developed using Java and object-oriented programming principles.

## Description
The system stores, processes, and reports academic data including students, faculty members, courses, departments, and programs.
It reads data from multiple input files and generates structured, human-readable reports.

## Features
- Student and faculty management
- Course and department handling
- File-based data processing
- Academic reporting

## OOP Concepts Used
- Inheritance
- Encapsulation
- Polymorphism
- Abstraction
