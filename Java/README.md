# ☕ Java Projects Collection

This directory contains a collection of Java projects developed during my Computer Engineering coursework, focusing on robust software design, object-oriented programming (OOP) principles, and advanced data structures and algorithms.

## 📂 Projects Overview

* **[MediPlan: Intelligent Diagnostic Planner](./hospital-test-evaluation-system)**
  An automated diagnostic planning system that utilizes Top-Down and Bottom-Up Dynamic Programming (DP) alongside Topological Sorting to optimize hospital test execution and minimize patient physical burden.

* **[The Aetheria Project: Smart City Architect](./smart-city-project)**
  A network architecture system simulating a smart city. It implements Minimum Spanning Trees (MST) for power grids, Topological Sorting with Cycle Detection for system boot sequences, and Breadth-First Search (BFS) for bio-leak containment routing.

* **[AstraNova Spaceport: Launch & Transit Planner](./spaceport-launch-preparation)**
  A dual-purpose scheduling and routing system. It uses graph-based prerequisite scheduling for launch operations and shortest-path algorithms to optimize transit routes, utilizing robust Regular Expressions (Regex) for input parsing.

* **[University Library Management System](./university-library-management-system)**
  An OOP-based system simulating a university library environment. It manages distinct user roles, enforces borrowing rules, handles penalty calculations, and utilizes file-based input processing.

* **[University Student Management System](./university-student-management-system)**
  A management architecture designed to handle student data, academic records, and university administrative tracking using core object-oriented design patterns.

* **[Zoo Manager](./zoo-manager)**
  A system heavily focused on applying deep OOP concepts—such as inheritance, encapsulation, and polymorphism—to simulate distinct animal classifications, behaviors, and habitat management.

* **[Tank Game](./tank-game)**
  A Java-based application that puts practical object-oriented design, logic flows, and interactive elements into action.

## 🧠 Core Topics & Skills Covered

* **Object-Oriented Programming (OOP):** Deep application of Inheritance, Encapsulation, Polymorphism, and Abstraction to create modular and reusable code.
* **Algorithm Design:** Implementation of advanced concepts including Dynamic Programming (Memoization and Tabulation), Graph Traversal (DFS/BFS), Shortest Path, and Topological Sorting.
* **Data Management & Parsing:** Extensive use of File I/O processing, XML Parsing (`javax.xml.parsers`), and flexible data extraction using Regular Expressions (`java.util.regex`).
* **System Robustness:** Comprehensive exception handling, edge-case mitigation, and input validation.
* **User Interfaces:** GUI development with JavaFX.
