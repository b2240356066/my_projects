# Zoo Manager

A Java application that simulates the management of a zoo using object-oriented programming principles.

## Description
This project makes use of abstraction and polymorphism to model animals and zoo operations.
Custom exceptions are defined and handled to ensure robust system behavior.

## Features
- Animal hierarchy and abstraction
- Polymorphic behavior
- Custom exception handling

## OOP Concepts Used
- Abstraction
- Polymorphism
- Inheritance
- Exception handling
